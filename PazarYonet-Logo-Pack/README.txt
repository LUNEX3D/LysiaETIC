PazarYonet — Logo ve Marka Varlıkları Paketi
============================================

Web: https://pazaryonet.com
İletişim: info@pazaryonet.com

KLASÖR YAPISI
-------------
svg/          — Vektör kaynak (sınırsız ölçek)
png/ikon/     — Kare uygulama ikonları (PWA, favicon, mağaza)
png/yatay/    — Yatay tam logo (SVG, tarayıcıda açılabilir)
png/favicon/  — Sekme ikonu

RENK PALETİ
-----------
Mor gradient arka plan:
  #5b21b6  (sol üst)
  #7c3aed  (orta)
  #4c1d95  (sağ alt)
Sembol beyaz:     #FFFFFF
Accent çizgiler:  #f5f3ff → #ddd6fe
Ok:               #c4b5fd
Kenarlık:         rgba(255,255,255,0.14)

İKON SEMBOLÜ
------------
- Yuvarlatılmış kare (rx ≈ 26 / 120 birim)
- Beyaz "P" harfi + sağda 3 liste çizgisi + sağ altta büyüme oku

YAZILI LOGO (pazaryonet-logo-full.svg)
--------------------------------------
- PAZARYONET — 56px, ExtraBold (800)
- Alt yazı: PAZARYERİ YÖNETİM PLATFORMU — 14px, #8b85a8

KULLANIM
--------
- Baskı / büyük ekran: svg/pazaryonet-logo.svg veya png/ikon/icon-512x512.png
- Web header: svg/pazaryonet-logo-full.svg
- Favicon: png/favicon/favicon.png veya png/ikon/icon-32x32.png

PNG yeniden üretmek (geliştirici):
  cd frontend && npm run icons:generate
