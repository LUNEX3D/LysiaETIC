# İkas × Shopify — Araştırma Özeti (UI + IA)

## İkas paneli

| Alan | Davranış |
|------|----------|
| **Satış kanalları** | Sol alt / kanal listesi; her alan adı = ayrı vitrin |
| **Temalarım** | Aktif tema büyük önizleme, “Yayındaki Tema”, “Hazır”, **Temayı Özelleştir** (birincil CTA), **Tema Ekle** |
| **Tema editörü** | İçerik modu: modül sırası sürükle-bırak, üst/alt menü renkleri, banner |
| **SEO ve alan adı** | Google önizleme, meta, domain bağlama, yönlendirme |
| **Ana sayfa** | Kanal filtresi, tarih, KPI, grafik |

Kaynak: [İkas destek — tema düzenleme](https://support.ikas.com/tr/start-paket-rehberi-kombos-temami-nasil-duzenlerim), [İkas Studio](https://builders.ikas.com/docs/ikas-studio)

## Shopify Admin (2025)

| Alan | Davranış |
|------|----------|
| **Online Store → Themes** | Aktif tema kartı, Customize, tema kütüphanesi |
| **Content → Menus** | Menü yönetimi (admin’de ayrı) |
| **Theme editor** | Sol: bölüm ağacı, orta: canlı önizleme, sağ: ayarlar; cihaz toggle üstte |
| **Home** | Kurulum kartları, metrikler |

Kaynak: [Shopify Help — Menus](https://help.shopify.com/en/manual/online-store/menus-and-links), [Theme editor 2025](https://saleshunterthemes.com/blogs/shopify-themes/shopify-theme-editor)

## LysiaETIC V6+ eşlemesi

| Rakip | Lysia |
|-------|-------|
| Temalarım | `ec-wb-my-themes` → `EcommerceIkasThemesPage` |
| Temayı Özelleştir | `ec-wb-editor` + `ec-wb-design-studio` |
| Tema Ekle | `ec-wb-marketplace` |
| SEO ve alan adı | `ec-wb-seo` + `ec-wb-domain` |
| Satış kanalı seçici | `ec-stores` picker |
| Kurulum rehberi | `EcommerceSetupChecklist` (ana sayfa) |
